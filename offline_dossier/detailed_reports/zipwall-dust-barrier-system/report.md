## IBS Description
The ZipWall® Dust Barrier System can be set up in minutes without ladders or damage. Using spring-loaded, telescoping, twist-locking poles, a ZipWall barrier can be built as wide as needed and up to 20’ high. The system includes accessories to create doors in the barrier, seal off doorways, and form tight seals at the ceiling, floor, and walls. Barriers can be built using plastic sheeting or ZipSheet™ reusable barrier sheets. ZipWall® is ideal for contractors doing remodeling, demolition, painting and more.
